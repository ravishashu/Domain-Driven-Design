Understanding the Core Concepts


Domain: The subject area for your application.
Entity: An object that has an identity.
Value Object: An object that does not have a unique identity.
Aggregate: A cluster of domain objects that can be treated as a single unit.
Repository: A mechanism for encapsulating storage, retrieval, and search behavior.
Service: A set of operations offered as an interface that stands alone in the model, without encapsulating state.
